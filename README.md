# Profile Intelligence Dashboard (Skeleton)

Initial skeleton for a profile-centric dashboard.

## Layout
- Fixed left profile summary panel (30% width on desktop)
- Right content workspace (70%) with domain tabs

## Tabs
1. Profile Summary
2. Case & Enforcement History
3. Interactions & Associations
4. Mobility & Border Movement
5. Vehicles & Traffic Compliance
6. Maritime Assets
7. Business Links

## Run
Open `index.html` directly in a browser.
